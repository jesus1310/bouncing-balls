Pasos a seguir para realizar esta actividad
-------------------------------------------

1º El usuario indica el número de bolas que se crearán.
2º  a- Las bolas tendrán un radio aleatorio.
    b- Las bolas tendrán un color aleatorio.
    c- Las bolas se crean en una posicion aleatoria (a la izquierda de la ventana)
3º Si una bola se sale por la derecha finalizará la aplicación
